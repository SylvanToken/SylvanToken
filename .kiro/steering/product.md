# Product Overview

## SylvanToken (Enhanced Sylvan Token - ESYL)

A production-ready, security-enhanced BEP-20 token built on Binance Smart Chain with advanced tokenomics, comprehensive security mechanisms, and sophisticated token locking/vesting systems.

### Core Features

- **Fixed Supply**: 1 billion tokens (1,000,000,000 ESYL)
- **Universal Fee System**: 1% transaction fee on all non-exempt transfers
- **Fee Distribution**: 50% operations, 25% burn (deflationary), 25% donations
- **Advanced Vesting**: Separate mechanisms for admin wallets (10% immediate + 90% vested over 20 months) and locked wallets (100% vested over 34 months)
- **Proportional Burning**: 10% of each vested release is burned, 90% goes to beneficiary
- **Dynamic Fee Exemptions**: Configurable exemption system with permanent and temporary categories

### Security Focus

The project emphasizes multi-layer security with comprehensive testing (163+ security tests), reentrancy protection, input validation, emergency controls, and access management. All critical operations are owner-restricted with cooldown periods.
