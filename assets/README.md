# Sylvan Token Assets

This directory contains brand assets for Sylvan Token (SYL).

## Logo Files

### Primary Logo
- **File:** `images/sylvan-token-logo.png`
- **Format:** PNG with transparency
- **Colors:** Blue gradient (top) to Green gradient (bottom)
- **Symbol:** Stylized "S" with geometric diamond pattern
- **Usage:** Primary logo for all official materials

### Logo Specifications

#### Colors
- **Primary Blue:** #00B4D8 (Cyan/Turquoise)
- **Secondary Blue:** #0077B6 (Deep Blue)
- **Primary Green:** #52B788 (Forest Green)
- **Secondary Green:** #2D6A4F (Dark Green)

#### Sizes
- **Large:** 1024x1024px (for websites, social media)
- **Medium:** 512x512px (for documentation)
- **Small:** 256x256px (for icons)
- **Favicon:** 32x32px, 16x16px

## Usage Guidelines

### Where to Use

1. **README.md** - Project header
2. **Documentation** - All docs headers
3. **Website** - Favicon and header
4. **GitHub** - Repository social preview
5. **Package** - npm package icon
6. **Presentations** - Marketing materials

### Do's
✅ Use on white or dark backgrounds
✅ Maintain aspect ratio
✅ Keep minimum size of 32x32px
✅ Use PNG format for transparency

### Don'ts
❌ Don't distort or stretch
❌ Don't change colors
❌ Don't add effects or filters
❌ Don't use low-resolution versions

## Brand Identity

### Concept
The Sylvan Token logo represents:
- **Circular Flow:** Continuous ecosystem and sustainability
- **Blue Gradient:** Technology, trust, and innovation
- **Green Gradient:** Nature, growth, and environmental focus
- **Geometric Diamond:** Precision, value, and blockchain structure
- **S Symbol:** Sylvan identity with modern design

### Symbolism
- **Waves/Circles:** Natural cycles and token circulation
- **Diamond Pattern:** Blockchain blocks and digital assets
- **Color Transition:** Bridge between technology (blue) and nature (green)
